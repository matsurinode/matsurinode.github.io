Q1：Surfboard下载后打不开，怎么办？

答：请检查是否来自官网下载安装；如为macOS，请进入“系统偏好设置 > 安全性与隐私”，允许打开来自未认证开发者的应用。

Q2：Surfboard和Shadowrocket有何区别？

答：两者功能相似，但Shadowrocket仅限App Store购买使用，而Surfboard提供更多自由下载方式，并在规则配置与日志功能上更友好，且支持导入Clash格式配置。

Q3：Surfboard是免费的吗？

答：iOS版本为免费应用，部分高级功能可能需内购解锁；macOS版本通常完全免费使用。这相较许多一次性收费工具显得更具性价比。

Q4：如何添加订阅或导入节点？

答：可在配置界面点击“添加配置”，输入机场提供的订阅链接或手动粘贴JSON格式配置文件，支持一键导入。

Q5：Surfboard安全吗？

答：只要你从官网下载安装，并使用可信节点（如购买自用机场），Surfboard本身是完全安全的，并不收集或记录用户数据。

FAQ

QuantumultX 可以按协议类型分流吗？
支持。用户可按 TCP、UDP 或特定应用协议分流流量，结合策略组，实现游戏加速、视频优化或广告屏蔽等定制化网络策略，提高访问速度和稳定性。
Trojan TLS 伪装如何降低被封锁风险？
通过选择真实可访问的域名并安装有效证书，Trojan 流量可完全模拟 HTTPS 请求。这样可隐藏流量特征，降低被封锁和 DPI 检测的可能性，保证节点长期可用。
Mihomo 如何进行远程控制管理？
可在配置中启用 external-controller 并绑定管理端口，使用 WebUI 或 Mihomo Dashboard 登录。可远程查看节点状态、切换代理、编辑规则，非常适合在服务器上集中管理多个客户端。
Mihomo 是否比 Clash Meta 更快？
Mihomo 是 Clash Meta 的优化分支，采用最新 Go 语言版本编译并针对多线程性能进行了改进。整体运行效率略高，特别是在多节点切换和 DNS 分流方面表现更流畅。
如何在 Clash Premium 中使用 TUN 模式加速游戏？
Clash Premium 版本支持系统级 TUN 模式，可将所有 UDP 流量转发至代理节点。启用该模式后需管理员权限，并在配置文件中添加 dns 与 tun 模块，选择低延迟节点即可实现游戏加速。
SSR 节点和 V2Ray 节点可以混合使用吗？
可以，但要使用支持多协议的客户端，如 Clash、Sing-box、Mihomo 等。这类客户端能在同一配置文件中加载不同协议的节点，通过规则分流让 SSR 与 V2Ray 同时工作。
Hiddify Next 如何管理用户流量配额？
管理员可以为每个用户设置上传和下载带宽以及总流量上限。超出限制可自动降速或阻止访问，保证服务器资源合理分配，提高整体网络公平性和用户体验。
Potatso Lite 能否手动添加节点？
Potatso Lite 支持手动添加 Shadowsocks 节点。用户在“配置”中选择添加节点，输入服务器、端口、加密方式等信息即可建立连接，适合手动管理单个线路。
Sing-box 可以替换 Clash 配置文件吗？
Sing-box 支持直接解析 Clash YAML 配置，并转换为自身 JSON 格式。节点、策略组和规则基本兼容，但部分自定义脚本可能需要手动调整才能完全使用。
Mihomo 是什么？
Mihomo 是 Clash Premium 的开源替代项目，兼容原版 Clash 配置文件并扩展了部分新功能。它性能优异、内存占用低，适用于希望自定义构建代理环境的高级用户。
